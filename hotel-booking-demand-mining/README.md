# Hotel Booking Demand – Data Mining Project

## 1. Dataset
Hotel Booking Demand dataset contains booking information for a city hotel and a resort hotel.

Source: https://www.kaggle.com/datasets/jessemostipak/hotel-booking-demand

## 2. Research Objectives
This project aims to apply data mining techniques to answer the following research questions:

- Classification: Can we predict booking cancellation?
- Clustering: Can we segment customers based on booking behavior?
- Time Series: How does booking demand change over time?
- Association Rules: What patterns exist among booking attributes?

## 3. Techniques Used
- Classification: Logistic Regression, Random Forest, XGBoost
- Clustering: K-Means, Hierarchical Clustering, DBSCAN
- Time Series: Decomposition, ARIMA, Prophet
- Association Rules: Apriori, FP-Growth
